Help & how it works
📄 Pages — two views
The Pages tab has two sub-views you switch between at the top:
  • Expanded view — original expandable rows grouped by category. Click any row to see all keywords for that page.
  • Ahrefs-style table — flat table mirroring Ahrefs Top Pages: URL, Page type, Status (New/Lost/↑/↓), Traffic with bar, Change %, Keywords, KW change, Top keyword, estimated Volume, and inline keyword dropdown showing top 15 keywords.
ⓘ Volume is estimated from GSC impression/CTR data. UR and Ref. Domains require the Ahrefs API.
🆕 Team sync via Supabase
Connect a Supabase project so one person imports ZIPs and the whole team sees the same snapshots.
  • Go to Settings → Team sync, enter your Supabase URL + anon key, click Connect
  • On the Imports page, a ↑ Sync button appears next to each local snapshot
  • Team members see synced snapshots with a 🆕 Team badge — pulled automatically on page load
⚠ You need to create the insiderank_snapshots table first. SQL is in Settings → Team sync.
📊 Why GSC instead of Ahrefs?
Ahrefs only crawls the top ~30 results. Anything ranking 31–100 shows as "Lost" in Ahrefs. GSC reports actual impressions across the full SERP.
💾 Data storage
Without Supabase: all data stays in your own browser. With Supabase: snapshots stored in your own database — Inside Rank never stores your data.

ALTER TABLE insiderank_snapshots ENABLE ROW LEVEL SECURITY;
